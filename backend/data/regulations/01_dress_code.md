# Дресс-код Turonbank

## Общие принципы

Сотрудники Turonbank поддерживают деловой стиль одежды. Внешний вид
должен подчёркивать профессионализм и надёжность банка.

## Для сотрудников фронт-офиса

- Деловой костюм классических цветов: тёмно-синий, серый, чёрный
- Рубашка/блуза светлых тонов
- Закрытая обувь, начищенная
- Бейдж с именем и должностью носится на видном месте

## Для бэк-офиса

- Допускается smart casual: брюки/юбка делового кроя, рубашка/блуза, пиджак опционально
- Кроссовки, джинсы рваные/яркие — запрещены

## Запрещено

- Спортивная одежда (вне корпоративных мероприятий)
- Открытая обувь и пляжная обувь
- Яркие принты с надписями
- Чрезмерное количество украшений

## Пятница

По пятницам допускается smart casual для всех сотрудников, кроме
сотрудников VIP-обслуживания и руководителей переговорных встреч.
