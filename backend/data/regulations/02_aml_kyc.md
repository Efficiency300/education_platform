# AML/KYC: Идентификация клиента

## Когда требуется полная идентификация

- При открытии любого счёта (физическое и юридическое лицо)
- При операциях свыше эквивалента 100 млн UZS
- При установлении долгосрочных деловых отношений

## Что проверяется

1. Документ, удостоверяющий личность (паспорт, ID-карта)
2. ИНН/ПИНФЛ
3. Источник происхождения средств для крупных операций
4. Проверка по санкционным спискам через АБС

## Запрещено

- Открывать счёт без оригинала документа
- Принимать копии без сверки с оригиналом
- Пропускать заполнение KYC-анкеты
- Открывать счёт лицам из санкционного списка

## Особые случаи

- При подозрении на сделки, связанные с легализацией доходов,
  сотрудник обязан немедленно сообщить в Compliance-департамент.
- KYC-анкета должна обновляться не реже одного раза в 3 года.
